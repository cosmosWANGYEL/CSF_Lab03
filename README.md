# CSF_Lab03
Aim: To develop a clear understanding on control structures of Python by implementing conditional statements(if, elif, and else), loop(for and while), break and continue statements, and functions.

List of programs:
1. To check numbers(+ve, -ve or zero)
-> Checking whether the numbers are positive, negative or zero by implementing if-else statements and comparison method.
2. To check even or odd number
-> Checking if the number is even or odd by implementing modulus operator.
3. Largest between two numbers
-> Checking numbers using if-else and comparison method.
4. Largest among three numbers
-> Checking largest among three numbers using nested if-else statements because there are three variables so we need to compare all of them to get the largest among them.
5. Grade comparison
-> Giving the grade according to their score by using if-else and comparison method.
6. Simple calculator
-> Making calculator using arithmetic operators, comparison operator and if-else statements.
7. Printing numbers(1-10)
-> By using for loop, we generated numbers from 1 to 10.
8. Printing even numbers(1-20)
-> By using for loop and modulus operator, we generated a numbers from 1 to 20 but only even numbers included.
9. Sum of first ten numbers
-> We get sum of first 10 numbers using increment method that adds up till 10.
10. Multiplication table
-> Generation of multiplication table using for loop.
11. Printing numbers(1-5)
-> Using while loop, we generated numbers from 1 to 5 by setting conditions.
12. Sum of numbers from 1 to n
-> Sum of numbers by using increment method that adds up till nth term(eg. 1+2+.....+n)
13. While loop(zero)
-> We made a program that keeps running until the user enters 0.
14. Using break to generate output
-> By using loop, then break after the required output.
15. using continue to generate output
-> By using loop, then continue after getting the required output.
16. Targetting in a list
-> Using list and then targetting the numbers in a list, using loop and if-else statements to find the numbers in a list.
17. Printing "Hello, world!"
-> To print "Hello, world!" by defining the function.
18. To find the square of a number
-> To get the square of a number by defining the function(eg. 6 x 6 = 36)
19. To add numbers using a function
-> Adding two numbers by defining the function.
20. To check even or odd using function
-> Checking numbers using modulus operator and if-else statements by defining a function.
21. To find factorial of a number
-> Calculating the factorial of a number by using if-else statements after defining a function.
22. Generating multiplication table
-> Generating multiplication table by using loop after defining a function.
23. Even or odd function
-> Checking the number by using if-else and loop after defining the function.
24. Calculator
-> Generating calculator by using while loop and if-else statements after defining the operator function.
